Acerca de esta plataforma

Objetivo educativo

Esta plataforma ha sido diseñada para proporcionar capacitación técnica al personal de hemoterapia sobre el uso correcto y eficiente del Sistema automático de recolección de componentes sanguíneos Trima Accel®.

A través de un enfoque estructurado y secuencial, los profesionales podrán adquirir las competencias necesarias para operar el equipo de manera segura y efectiva.

Metodología

La formación se basa en un enfoque modular que combina:

  • Contenido teórico con imágenes de apoyo
  • Demostraciones visuales paso a paso
  • Ejercicios interactivos de verificación
  • Evaluaciones para confirmar la comprensión

Cómo utilizar esta plataforma

  1. Comienza por el Módulo 1 y avanza secuencialmente
  2. Lee detenidamente el contenido teórico de cada módulo
  3. Realiza los ejercicios interactivos para reforzar el aprendizaje
  4. Completa la evaluación al final de cada módulo
  5. Debes obtener al menos un 80% en cada evaluación para desbloquear el siguiente módulo
